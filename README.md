## Документация


> #### Запуск проекта
1. docker-compose build
2. docker-compose up

> #### Полезные команды:
* sudo chown -R $USER /home/user/PycharmProjects/wb_api/data

> #### Адреса:

1. /api/v1/articles/
> * все артикулы
2. /create/art/
> * добавить артикул
3. /create/file/
> * загрузить файл (xlsx) с артикулами (артикулы должны вводиться построчно в первой колонке)
